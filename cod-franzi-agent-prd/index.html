Locked Decisions

Posture
Augment Franzi (draft → Slack approve → send). Not autonomous. Russ's "real human gave a damn" doctrine is the value prop.
Channel Routing
Same agent, both channels: Instagram DMs (native — matches Kelsey + Tanguy "DM Setter" prompt) + Bison email replies (same playbook, adapted formatting). PostHawk for outbound (built-in follow-up reminders). GHL fires only post-booking.
Follow-up Cadence
PostHawk's native reminders own the schedule. No cron in our agent. Each touch drafted fresh when PostHawk webhook fires.
Architecture Pattern
Approach B — Reply Drafter + Winning-Pattern Engine. Mirrors existing cpg-attio/src/agents/reply-agent.ts pattern.
Voice Grounding
Two-layer: franzi_voice.md (Kelsey + Tanguy canonical prompt, always loads) + winning_patterns.json (extracted from Bison history, situational citations).
Models
Haiku 4.5 classifier · Opus 4.7 for booking-intent + objections + qualified-ready · Sonnet 4.6 for routine qualifying + cold-opener responses · No draft for hard-stop / OOO / unsubscribe.
Build Owner
TBD — PRD written for production quality, decision after Russ + Franzi review.
Cost Strategy
Aggressive prompt caching (90% discount on cache reads) + tiered model routing + Anthropic Batch API for offline pattern extraction. Target: $30–60/mo metered API at 50 drafts/day. Claude Code Max for dev/ramp-up (free against subscription).
