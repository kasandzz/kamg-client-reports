Cloud Function Gold Standard Pattern:
  • Gen 2, Node.js 22, self-contained index.js
  • WRITE_TRUNCATE (full reload each run) -- no incremental complexity
  • Env vars for all secrets, zero local file dependencies
  • Stream to /tmp NDJSON for large datasets (Stripe pattern)
  • Rate limiting, retry logic, pagination support
  • ETL run logging to etl_run_log table
