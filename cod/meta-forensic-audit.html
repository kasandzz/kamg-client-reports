Warning Interest Stacking Violation (Post-Andromeda)

172 ad sets use interest-based targeting, many with "Interest Stack" in their names. Post-Andromeda (Meta's ML overhaul), interest stacking is deprecated. Meta's algorithm now optimizes better with broad targeting + strong creative signals.

  • The attorney niche ($201 CPA, $78.70 CPM) uses broad + niche call-out -- proving the model
Recommendation
Migrate all interest stacks to broad + niche creative. Keep interests only as a signal layer in CBO campaigns where Meta can use them as a suggestion, not a restriction.

Structural Diagnostics

Critical ABO vs CBO -- Suboptimal Post-Andromeda Structure

The vast majority of campaigns use ABO. Post-Andromeda, CBO is the preferred structure because it allows Meta's ML to dynamically allocate budget.

  • ABO campaigns: ~90% of active spend
  • CBO campaigns: Only 2 active (Licensed Therapists at $217 CPA, Coaches at $197 CPA)
  • CBO campaigns are outperforming ABO by 30-40% on CPA
Piliero M3 Framework
"CBO is not magic, but it is superior if structured correctly." Ideal structure: Prospecting CBO + ASC + Retention (3-layer model). Each CBO should have 3-5 ad sets with distinct audiences.
Critical Campaign Proliferation -- 147 Campaigns is Extreme Fragmentation

147 campaigns with only 5 active means 142 campaigns were created, ran, and paused without consolidation:

  • Auction fragmentation: Multiple campaigns targeting the same audiences bid against each other
  • Learning phase waste: New campaigns reset the learning phase (~50 conversions per ad set per week)
  • Data fragmentation: Conversion data spread across 147 campaigns instead of being concentrated
Recommendation
Consolidate to 3-5 campaign buckets: (1) Cold Prospecting CBO, (2) Niche Industry ABO with cost caps, (3) Advantage+ Shopping, (4) Warm Retargeting, (5) Authority/Engagement.
Critical Retargeting Collapse -- $31K Spent, $2,108 CPA

The "Retargeting - VSL Book Call" campaign has spent $31,619 with only 15 purchases at $2,108 CPA. The single biggest money leak.
