VIP TY Page: Booking Setup

The only remaining piece • April 23, 2026

What We're Solving

The "Book Your Call Now" CTA on the VIP Thank-You page should NOT let people book immediately. They need to attend the workshop first. So booking should only be possible ~27 hours after purchase. We need to do this without breaking the existing sales team calendar or email triggers.

Why 27 hours?

Workshop is usually the next day. 27 hours gives them time to watch it before booking a call. We don't want people booking calls before they've seen the content.

Recommended - Easiest

Option A: New GHL Calendar with Minimum Notice

1
Duplicate the existing sales team calendar
Go to GHL Calendars. Find the current sales team booking calendar. Click the 3 dots and Clone it. Name the clone something like:
VIP Post-Workshop Booking
2
Set Minimum Scheduling Notice to 27 hours
In the cloned calendar settings, find Minimum Scheduling Notice (sometimes called "Buffer before booking" or "Minimum advance notice").

Set it to 27 hours (or 1 day + 3 hours if it only takes days/hours separately).

This means: when someone opens the booking page, the earliest slot they can see is 27+ hours away. They literally cannot book anything sooner.
3
Keep same team members / availability
Make sure the same sales reps are assigned to this calendar with the same availability windows. We're only changing the minimum notice, nothing else.
4
Get the new calendar booking link
Copy the booking link from the cloned calendar. It will look something like:
https://api.leadconnectorhq.com/widget/booking/XXXXXXXXX
5
Update the CTA on VIP TY page
On cod-hto-workshop-thanks-vip-b, update the "Book Your Call Now" button to point to the new calendar link from Step 4.

DO NOT edit the original sales team calendar. We're creating a separate one so nothing breaks for existing bookings or workflows.

6
Add new calendar to existing workflows (CRITICAL)
The cloned calendar won't automatically trigger existing workflows. You need to add it manually:

Step by step:
• Go to GHL Automation → Workflows
• Find every workflow that triggers on the original sales calendar (look for triggers like "Appointment Booked", "Appointment Reminder", "Appointment Status Changed", etc.)
• Open each one and click the trigger/entry condition
• Where it says "Calendar" or "Calendar is [original name]", add the new "VIP Post-Workshop Booking" calendar as an additional entry condition
• Usually this means changing "Calendar is [Original]" to "Calendar is [Original] OR [VIP Post-Workshop Booking]"
Workflows to check:
• Booking confirmation email
• Appointment reminder (24h, 1h, etc.)
• No-show follow-up
• Post-call follow-up
• Internal team notifications (Slack/email to sales rep)
• Any CRM pipeline/stage automations triggered by booking

How to find them fast: Search the workflow list for the original calendar name. Every workflow that references it needs the new calendar added too.

If you miss a workflow: Bookings from the new calendar won't trigger confirmations, reminders, or team notifications. The customer books but gets no email. Sales rep doesn't know about the call. Check every workflow.

7
Test it end-to-end
Open the new booking link in an incognito window. Confirm that:
• No slots show for today or early tomorrow
• The first available slot is 27+ hours from now
• The booking form works and submits correctly
• Confirmation email arrives after test booking
• Sales rep gets notified (Slack/email/CRM)
• Cancel the test booking after verifying
