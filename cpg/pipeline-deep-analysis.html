💡 Section 5: Suggestions and Tests STRATEGIC FIXES

1. Fix the Setup Fee -- This is P0

31 deals cite setup fee as the blocker. This is a commercial product problem, not a rep skill problem.
Tests to run:
-- Deferred fee model: Setup fee is charged from first 60 days of CPA earnings, not upfront. Reduces perceived risk to zero.
-- Tier-based pricing: Tier 1 brands get discounted setup ($1,500 instead of $2,000-3,000) in exchange for higher CPA commitment. Tier 3 must pay full rate (discouraging Tier 3 altogether).
-- Performance bond framing: Reframe the setup fee as a "performance fund" -- money held and applied against the first X conversions at the agreed CPA, then replenished. Brands see it as CPA budget, not a fee.
-- Current data: 31/336 deals = 9.2% of pipeline directly blocked by this issue.

2. Follow-Up Cadence -- Formalize the Process

The April 8-10 blitz shows the team CAN do mass follow-ups. The problem is it's not systematic.
-- Implement a 7-day / 14-day / 30-day follow-up cadence for all "Meeting Completed" deals.
-- Any deal with no contact in 90 days should be moved to "Dormant" and only contacted quarterly.
-- Build a simple tracking system so the blitz becomes weekly, not quarterly.
-- Current gap: ~150 deals in pipeline with no 2026 Last FU recorded.

3. No-Show Recovery Protocol

38 no-shows = 38 wasted booking slots. A 24-hour reminder + same-day reschedule prompt would recover an estimated 30-40%.
-- Send a 24h text + email reminder for every booked call.
-- If no-show occurs, send a reschedule link within 1 hour.
-- Track no-show by rep -- Juli's 15.4% rate vs Krik's 7.1% suggests different prospect quality or booking channels.
-- If recovering 10 no-shows at 16.7% meeting-to-signed rate = 1.7 additional clients.

4. ICP Refinement -- Stop Taking Tier 3 Deals

Tier 3 close rate: 1.6%. 61 deals for 1 signed client. This is a resource drain.
-- Implement a hard minimum: 10,000+ monthly traffic, $50+ AOV, minimum $1,500 test budget.
-- Tier 3 prospects should be declined at qualification rather than booked for calls.
-- If redirected to Tier 1 leads at a 16.7% close rate, those 61 Tier 3 deal slots could theoretically yield 10 additional signed clients.
-- Health, Beauty, Food/Beverage are the top-performing verticals. Double down on those.

5. Qualification Checklist Before Booking

Add 3 knockout questions before any call gets booked.
-- "Are you open to a performance partnership with a small upfront setup investment?" -- kills 9% of pipeline waste before the call.
-- "What is your monthly website traffic and average order value?" -- filters Tier 3 at source.
-- "Are you the decision maker or would others need to be involved?" -- sets expectation for CEO calls when needed.
-- Estimated impact: 15-20% reduction in dead-end calls, more time for qualified prospects.

6. Krik-Close Protocol for Big Deals

Krik closes at 25.9%. Use Krik as a second-touch closer on high-value Juli deals.
-- For Tier 1 deals that have been in "Meeting Completed, Likely to Sign" for 30+ days without conversion, introduce Krik as a "senior partner review" call.
-- This adds a new touch, a higher-authority voice, and resets the prospect's engagement.
-- Prioritize: Juvenon, Happy Hippo, Wonder Project, Statik, Essence/Essence Ring -- all Likely to Sign with 60+ day staleness.

7. Referral Program -- Leverage Signed Clients

43 signed clients. Have any been asked for referrals systematically?
-- Derma Dreams notes "referral from Stephen Ezell from Trully Free" -- this model works.
-- Implement a 60-day post-onboarding referral ask to all signed clients.
-- Even a 10% referral conversion rate on 43 clients = 4 inbound qualified leads for free.
-- Health and Supplements clients (14 signed) are the best network for referrals given category clustering.

8. Pipeline Hygiene -- Archive Dead Deals

~150 deals have no 2026 Last FU. These are clogging the active pipeline view.
-- Archive any deal with no contact in 180+ days that isn't in a scheduled ping status.
-- Create a "Dormant" stage for deals with 90-180 day gaps -- ping once per quarter, then archive if no response.
-- Clean pipeline = accurate close rate data = better sales decisions.

9. Apparel Vertical -- Fix or Exit (NEW)

24 Apparel and Fashion deals. 0 signed. Investigate why before booking more.
-- Review all declined apparel deals for common objections (Studio Seven, Boujee Hippie x2, Bondi Born, Postum, etc.).
-- Likely issues: fashion brands have lower margins, complex seasonal inventory, higher return rates making CPA math difficult.

10. Ads=No Brands Are Better Prospects (NEW)
